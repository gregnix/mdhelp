# Quick Start

Get started with mdhelp in five minutes.

## Launch

```
wish mdhelp.tcl ./my-folder
```

Without arguments, mdhelp opens the last used folder
or the built-in `docs/` directory.

## Window Layout

The main window has four areas:

**Toolbar** (top): Back, Forward, Home, Open, Find, Font Size, Edit, PDF.

**File Tree** (upper left): Shows all Markdown files as a tree.
Click to open a file. The current entry is highlighted.

**Table of Contents** (lower left): Shows the headings of the current file.
Click to jump to a heading. Indentation indicates the level.

**Viewer** (right): Shows the rendered Markdown document with tables,
code blocks, images and links.

**Status Bar** (bottom): Shows the file path and font size.
When hovering over a link, the target URL appears.

## Key Actions

**Navigate:** Click on files in the tree or on links in the text.
Use the arrow buttons or Alt+Left/Alt+Right to go back and forward.

**Search:** Ctrl+F opens the search bar. Choose "Page" for the
current file or "All Files" for a global search.
F3 jumps to the next match.

**Edit:** Ctrl+E opens the integrated editor with live preview.
Ctrl+S saves. The viewer updates automatically.

**Adjust Font:** Ctrl+Plus enlarges, Ctrl+Minus shrinks.

**Create PDF:** Click "PDF" in the toolbar or choose
File > Export PDF.
